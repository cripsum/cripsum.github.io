## About

Cripsum is a personal web project that slowly evolved into a bigger platform.

It started as a simple idea and turned into a mix of:

- customizable profile pages
- account systems
- achievements and missions
- gacha system
- games
- social features
- experiments
- random ideas implemented at 3am

The project keeps changing over time.  
New systems and features get added whenever I feel like building something cool.

---

## Features

- 🎨 Fully customizable profiles
- 🏆 Achievement and daily / weekly missions system
- 📦 Pull system with inventory
- 🌍 Multi-language support
- 🔐 Authentication system
- ⚡ Lightweight frontend
- 🧪 Experimental features
- 💬 Social/profile interactions
- 📱 Responsive UI
- 🚀 Constant updates

## Philosophy

This project isn't trying to be a polished corporate product.

I mostly build things because:

- they're fun to make
- I want to learn something new
- or I randomly got an idea at night

Some systems are serious.  
Some are experimental.  
Some probably shouldn't exist.

That's part of the project.
